RxDeviceSetup

Response from controller transporting the currently used configuration: ODR, Range and Scale. This package is received at the end of stream.

RxFifoOverflow

Response from controller indicating that the acceleration sensor’s Fifo could not be consumed/read in time, thus an overrun occurred.

RxBufferOverflow

Response from controller indicating that the circular buffer overflowed.

RxTransmissionError

Response from controller indicating that the transmission to the host was erroneous (while sampling).

RxSamplingStarted

Response from controller indicating that sampling has been started just before this response was issued. This package is received at the start of stream.

RxSamplingStopped

Response from controller indicating that the sampling has been stopped (for whatever reason).

RxSamplingFinished

RxSamplingAborted
